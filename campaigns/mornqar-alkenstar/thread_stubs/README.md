# Thread Stub Files

These are minimal thread JSON files created to satisfy the missing thread references
from the five current playerCharacter files.

They use a compact thread shape:

- id
- type
- title
- summary
- status
- threadType
- priority
- campaignId
- notes
- relations

They are intended as migration-safe starter files. Enrich them later with:
- description
- related events
- related effects
- stronger thread relations
- GM-facing progress or clue tracking
